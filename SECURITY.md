# Security

This repo is intended to be public-safe source code for the Suede Market Maker.

Never commit:

- `.env`
- private keys
- wallet JSON files
- funding wallet files
- Telegram bot tokens
- Telegram chat IDs
- dashboard logs
- trade ledgers
- generated runtime state

Use `.env.example` as the public template and keep real values only in your local `.env`.

If you need help with setup, reach out through [suedeai.ai](https://suedeai.ai).
